python asyncio + angular.js
===========================

Install
-------
    $ sudo npm install -g gulp bower
    $ npm install
    $ bower install
    $ ./bin/buildenv.py
    $ sudo ./bin/configure_nginx.py

Add to `/etc/hosts`:

    127.0.1.1   aio-angular.l.com

Run dev server:

    $ gulp

Open in browser: <http://aio-angular.l.com/>
